Project Summary

What this repository contains

  • train_resnet152_unet.py for training a 4-channel U-Net with a ResNet152 encoder.
  • analyze_brats_project.py for checkpoint-aware project analysis and Dice-style metrics.
  • generate_preprocessing_report.py for preprocessing visuals and dataset summaries.
  • BraTS 2020 training and validation folders with NIfTI volumes and metadata CSV files.

Model Snapshot

Training setup from the code

  • Architecture: U-Net from segmentation-models-pytorch.
  • Encoder: resnet152 with default ImageNet weights.
  • Inputs: t1, t1ce, t2, and flair.
  • Classes: background, label 1, label 2, and label 4 remapped to class 3.
  • Training data indexing uses tumor-only slices when building the dataset.

Ages range from 18.98 to 86.65. Resection labels in survival_info.csv: 119 GTR, 10 STR, and 107 NA.

Validation metadata

Ages range from 21.73 to 85.68. All 29 validation rows in survival_evaluation.csv are labeled GTR.

Preprocessing

What happens before training

  1. Load 3D NIfTI volumes for all four MRI modalities.
  2. Normalize each volume using nonzero voxels only.
  3. Select 2D slices and keep tumor-only slices for training.
  4. Resize image slices to 256 × 256.
  5. Remap BraTS label 4 to project class 3.
  6. Convert masks to four-class one-hot tensors for segmentation loss.

Evaluation

How results are reported

The analysis script looks for resnet152_unet_brats2020.pt. When that checkpoint exists, it reports mean Dice loss and per-class hard Dice values for background, label 1, label 2, and the remapped label4_as_3 class.

Important: the repository documentation explicitly notes that BraTS label 4 is shown as class 3 in project outputs.

How To Run

Commands from the project instructions

1. Create the environment

python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt

2. Windows Git Bash activation

source .venv/Scripts/activate

3. Small training run

python3 train_resnet152_unet.py --max-patients 1 --epochs 1 --batch-size 2 --num-workers 0

4. Save the expected checkpoint name

python3 train_resnet152_unet.py --max-patients 2 --epochs 2 --output resnet152_unet_brats2020.pt

5. Analyze the trained project

python3 analyze_brats_project.py

6. Generate preprocessing visuals

python3 generate_preprocessing_report.py --max-analysis-patients 24
